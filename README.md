# allennmsu-web-project
NMSU Allen Sikorski Web Development Project


## Project 2 - HTML Foundation Page

1. For this project I created the first version of my website using only HTML. The theme of the site is a trip through Southeast Asia, specifically the Philippines, Singapore, and Vietnam. The page highlights some of the places we visited such as the Chocolate Hills in Bohol, Gardens by the Bay in Singapore, and the Golden Bridge near Da Nang in Vietnam. 

2. While creating the page, I learned how to structure a webpage using the basic HTML layout with different HTML tags. I also learned how to link pages together using navigation links, how to insert images with the <img> tag, and how to organize information using lists and tables. This project helped me understand how a website is built starting from simple HTML before adding styling or interactivity.

3. One challenge I faced was making sure the file structure and links between pages were correct, especially linking the index page to the project2.html page inside the HTML folder. I also had to double check that images and external links worked properly. After testing and fixing a few small mistakes, I was able to get the pages to connect and display correctly.
