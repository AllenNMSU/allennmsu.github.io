# ICT 280 U71 Website Development Final Project

## Project Description

This is my final project website for GreenTech Solutions. GreenTech Solutions is a fictional company that focuses on environmental sustainability and technology.

The purpose of this website is to give small and medium sized businesses information about eco friendly technology, products, services, and resources. The site includes a home page, about page, products and services page, blog and resources page, and contact page.

## Live Site Link

GitHub Pages URL:

https://allennmsu.github.io/final/greentech-solutions/

## Features

### HTML

This project uses semantic HTML5 elements such as header, nav, main, section, article, form, and footer. Each page uses the same navigation menu so visitors can move between the pages easily.

### CSS

The website uses one external CSS file located in the css folder. The CSS is used to style the green color theme, page layout, navigation menu, sections, product cards, images, form, and footer.

### JavaScript

The website uses one external JavaScript file located in the js folder. JavaScript is used on the Products and Services page for the product search feature. It is also used on the Contact page to check the form before it is submitted.

### Responsiveness

The website is designed to work on desktop and mobile screen sizes. The images resize with the page, and the navigation changes on smaller screens to make it easier to use.

### Accessibility

The website includes basic accessibility features such as alt text for images, labels for form fields, readable colors, semantic HTML structure, and clear headings.

## How to Run

To view the website locally, open the index.html file in a web browser.

The website can also be viewed online using the GitHub Pages link above.
